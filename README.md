# BRV Business Intelligence

BRV Business Intelligence, BRV Software tarafindan gelistirilen, acik kaynakli bir is zekasi (Business Intelligence) ve raporlama uygulamasidir.

Bu repo, veri kaynaklarindan gelen verileri sorgulamak, gorsellestirmek ve gosterge tablolarinda (dashboard) birlestirmek icin kullanilan frontend ve backend kodunu icerir.

## Ozellikler

- Sorgu olusturucu arayuz ile SQL yazmadan sorgu tasarlama
- Cok cesitli grafik turleri (Number, Bar, Line, Row, Donut, Funnel, Table, Map)
- Calisma kitaplari, grafikler ve gosterge tablolarini organize etme
- Takim icinde paylasim ve yetkilendirme icin hazir alt yapi

## Gelistirme icin hizli baslangic

1. Depoyu klonlayin: `git clone https://github.com/cborova06/Business-Intelligence.git && cd Business-Intelligence`
2. Backend icin gerekli Python bagimliliklarini ve veritabani ortamini kendi altyapinizda kurun.
3. Frontend icin Node bagimliliklarini yukleyip derleyin: `cd frontend && yarn && yarn build` (veya gelistirme icin `yarn dev`).
4. Backend uygulamasini baslattiktan sonra tarayicidan ilgili URL uzerinden arayuze erisebilirsiniz.

Bu adimlar gelistirici ortam icindir. Uretim kurulumlari icin BRV Software tarafindan saglanan dokumantasyon ve destek takip edilmelidir.
